# ZRCLIENT

#### 介绍

Element主题风格的WPF管理后台系统，免费开源，欢迎下载点 ⭐

#### 软件架构

基于 .Net Framework 4.5.2 

UI框架：[HandyControl](https://gitee.com/handyorg/HandyControl)

图表组件：LiveCharts.Wpf

多样式引用：WpfMultiStyle

MVVM框架：MVVMLight

#### **开发工具**

IDE: Visual Studio 2019
